This is the TeddyBot, a unique bot to fuffil all your Discord needs. This isn't just a regular old Moderation Bot. It's a unique bot, designed to make your server the best with only one Discord bot. View the commands here.
This bot was made by Crevis, Chicken Master, and InvalidUser.











COMMANDS:

  • help - shows an embed that lists all the commands
  • about - shows an embed that displays the bots authors, uptime, ping, version, website, etc.
  • prefix - shows what the current prefix is; if you have the MANAGE SERVER permission, you can change it.
